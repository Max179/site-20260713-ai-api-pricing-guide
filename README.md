# AI API Pricing Comparison Guide 2026

A comprehensive guide comparing AI API pricing across major providers for developers.

## Features

- Compare LLM API pricing from OpenAI, Anthropic, Google, DeepSeek, Mistral, and more
- Real-time cost calculator for production workloads
- Filter by model tier, context window, and use case
- Up-to-date pricing data as of 2026

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
